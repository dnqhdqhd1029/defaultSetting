오프라인 환경에서 React Developer Tools 설치
Chrome 확장 프로그램 페이지 열기

Chrome 브라우저를 열고 주소 표시줄에 chrome://extensions/를 입력하여 확장 프로그램 페이지를 엽니다.
개발자 모드 활성화

확장 프로그램 페이지의 우측 상단에서 "개발자 모드"를 활성화합니다.
.crx 파일 설치

.crx 파일을 다운로드한 위치에서 파일을 드래그하여 확장 프로그램 페이지에 끌어다 놓습니다.
설치 확인 팝업이 표시되면 "추가"를 클릭하여 설치를 완료합니다.
확인 및 설정

설치가 완료되면, React Developer Tools 아이콘이 Chrome의 확장 프로그램 바에 나타납니다.
React 애플리케이션을 열고, React Developer Tools의 기능이 정상적으로 작동하는지 확인합니다.
3. 수동 설치 (선택사항)
만약 .crx 파일을 사용하지 않고, 소스 코드에서 직접 설치하고 싶다면, 다음과 같이 진행할 수 있습니다:

소스 코드 다운로드

React Developer Tools의 GitHub 리포지토리에서 소스 코드를 다운로드합니다. React DevTools GitHub 리포지토리에서 다운로드할 수 있습니다.
압축 해제 및 준비

다운로드한 소스 코드를 압축 해제한 후, 준비된 폴더를 확인합니다.
크롬에 로드

Chrome 확장 프로그램 페이지(chrome://extensions/)로 이동합니다.
"압축 해제된 확장 프로그램 로드" 버튼을 클릭합니다.
다운로드한 소스 코드의 경로를 선택하여 확장 프로그램을 로드합니다.
이 방법들을 통해 오프라인 환경에서도 React Developer Tools를 설치하고 사용할 수 있습니다.
